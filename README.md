# 如何编译?

1. 在项目目录打开终端
2. 执行`flutter pub get`
3. 把[build_release](./build_release)复制到终端执行

## 关于直连失效

1. 与运营商有关
2. 与地区有关
3. 可能间歇性的失效
4. 如果无法直连请开代理
